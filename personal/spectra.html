Spectra

Overview

Description: After taking Technical Game Development 1, where I developed a 2D ascii game engine in C++, I wanted to go back and try developing an engine again, fixing some of the mistakes I had made working on Aspect and using the knowledge I had gained in the class. Now that I was more comfortable with C++, I used that instead of Java. I also chose to learn and use Vulkan for this project, instead of OpenGL again. While this engine is not as fully-featured as Aspect, I was able to learn Vulkan well enough to support most of the same rendering capabilities. Despite being rather simple, I was able to use this engine for several projects in the class CS 4732 (Computer Animation). The features include:

  • OBJ mesh loading and rendering
  • GLSL shaders for fragment lighting, with one pass per light source
  • Component-oriented programming, similar to Unity
