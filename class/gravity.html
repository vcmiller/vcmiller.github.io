Accelerometer-Controlled Gravity

Demo Video

Overview

Description: A 2D platformer/action game where the player controls gravity by rotating an accelerometer. Other actions (movement and attacking) are done through conventional input. Measurements from the accelerometer are collected by an Arduino, which is connected to the computer via USB. In my Arduino code, I simply read the values and send them. In the game, a serial connection is opened, through which the accelerometer values are read. To make sure the controls are easy to use, other input is done via a joystick shield attached to the Arduino. Values from this device are sent alongside the accelerometer values.

Links: I am unable to provide the source, in accordance with WPI's policy on sharing source code for projects. Additionally, since this project requires an Arduino with my specific setup and code, providing an executable would be pointless.

Team: This was an entirely solo project.

Technologies Used:

  • Unity (with .NET serial and threading libraries)
  • Arduino Uno
  • ADXL335 Accelerometer
  • SparkFun Joystick Shield

Class: Novel Interfaces for Interactive Environments (IMGD 3100)

Assignment Requirements: This assignment was very open ended. We created various Arduino-based input devices throughout the term, and this project was meant to show what we learned.
